Button dropdowns

Use any button to trigger a dropdown menu by placing it within a .btn-group and providing the proper menu markup.

Plugin dependency

Button dropdowns require the dropdown plugin to be included in your version of Bootstrap.

<div class="btn-group">
  <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
    Action <span class="caret"></span>
  </button>
  <ul class="dropdown-menu" role="menu">
    <li><a href="#">Action</a></li>
    <li><a href="#">Another action</a></li>
    <li><a href="#">Something else here</a></li>
    <li class="divider"></li>
    <li><a href="#">Separated link</a></li>
  </ul>
</div>

Block style single button dropdown

Sometimes we need a block style single button dropdown. Add .btn-group-block-dropdown to the .btn-group class.

  • Mount Pleasant, SC
  • Sullivan's Island, SC
  • Isle of Palms, SC
<div class="btn-group btn-group-block-dropdown">
  <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
    <svg class="icon icon-location"><use xlink:href="#location"/></svg>  Charleston, SC <span class="caret"></span>
  </button>
  <ul class="dropdown-menu" role="menu">
    <li><a href="#">Mount Pleasant, SC</a></li>
    <li><a href="#">Sullivan's Island, SC</a></li>
    <li><a href="#">Isle of Palms, SC</a></li>
  </ul>
</div>
