Set to-do

Modular component for setting up a to-do task. Uses BoomQueries to apply size modifier classes.

BoomQuery Values

Breakpoint
Size Modifier Class
>= 420px
set-to-do--sm

Interaction note

When a set to-do date button is clicked, it should receive the .active class and change the displayed date accordingly.
When specific time is checked, the .hide class should be removed from .set-to-do__time__opts.
